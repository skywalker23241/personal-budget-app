# 个人预算记账助手

一个面向个人用户的网页端记账与预算管理 App。记录日常收支、管理月度预算、追踪贷款还款、统计收入、分析消费习惯（含衣食住行）、生成月度报表、追踪财务目标，并提供智能提醒与财务健康评分。

> 纯前端应用，**不依赖任何后端服务**，所有数据通过浏览器 `localStorage` 持久化，刷新或重开浏览器后依然保留。

---

## ✨ 技术栈

| 方面 | 选型 |
| --- | --- |
| 前端框架 | React 18 + TypeScript |
| 构建工具 | Vite 5 |
| 样式 | Tailwind CSS 3 |
| 图表 | Recharts 2 |
| 状态管理 | Zustand（含 `persist` 持久化中间件） |
| 路由 | React Router 6 |
| 数据存储 | 浏览器 localStorage |

UI 风格：简洁、现代、干净，柔和背景 + 卡片式布局，响应式适配桌面端与移动端。

## 🚀 如何安装与启动

### 环境要求
- Node.js ≥ 18（推荐 20/22）
- npm ≥ 9

### 1. 安装依赖
```bash
cd personal-budget-app
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```
启动后浏览器会自动打开 `http://localhost:5173`（若未自动打开，手动访问即可）。

### 3. 构建生产版本
```bash
npm run build      # 输出到 dist/
npm run preview    # 本地预览构建产物
```

### 4. 类型检查（可选）
```bash
npm run typecheck
```

> 首次打开时会自动载入一批**示例数据**（工资、餐饮、房租、交通、服装、车贷还款、车贷、储蓄目标、若干预算分类等），方便立即看到完整效果。可在「设置」中清空或重新载入示例数据。

## 🔮 后续可扩展功能建议

1. **后端与多端同步**：接入数据库（如 Supabase / Postgres）+ 账号体系，实现多设备同步与备份。
2. **PWA 离线安装**：增加 Service Worker 与 manifest，支持「添加到主屏幕」、离线可用。
3. **预算与提醒升级**：自定义分类、子分类、滚动预算、本地通知/邮件提醒、自定义记账周期真正驱动统计。
4. **贷款精算**：完整等额本息/等额本金摊销表、提前还款模拟、利息总额测算。
5. **更智能的分析**：基于历史的下月支出预测、订阅自动识别、消费习惯洞察与省钱建议（可结合 AI）。
6. **记账效率**：快捷输入、模板/常用记账、账单导入（微信/支付宝/银行 CSV 自动解析归类）、多币种与汇率。
7. **可视化增强**：自定义报表、年度总结、桑基图展示资金流向、目标里程碑。
8. **工程化**：单元测试（Vitest 覆盖 `lib`）、E2E 测试（Playwright）、深色模式、国际化（i18n）。

## 📌 数据安全提示

所有数据仅保存在**当前浏览器**的 localStorage 中。清除浏览器缓存、更换设备或浏览器会导致数据丢失。请在「设置 → 导出全部数据（JSON 备份）」中**定期备份**。
